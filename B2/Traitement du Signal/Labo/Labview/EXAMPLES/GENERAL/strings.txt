strings.llb contains following VIs:

Binary vs ASCII.vi:
	Shows the difference between ASCII data strings and binary data strings.

Build String.vi:
	 Shows how to use the various string functions to assemble a string
	 combining header information, a numeric value, and tailer information.

Extract Numbers.vi:
	Finds all numbers in the given string and puts them into an array as
	single-precision numbers.

Parse String.vi:
	 Shows how to use the string functions in LabVIEW to break an input string
	 into string subsets and to convert numeric ASCII strings to the actual
	 numeric values.

Search String and Replace.vi:
	Replaces all occurrences of a character in a string with a new character.
